## Design Context

### Users
ABO 面向科研工作者，尤其是研究生、博士和独立研究者。使用场景是本地化整理每日情报、把阅读输入转成研究推进、并在长期维度上持续观察自己的研究状态与节奏。

### Brand Personality
本产品的气质应当是：陪跑、清醒、可沉淀。界面不做夸张营销表达，而是像一套长期使用的个人研究操作台，让人感到可控、温和、但有推进感。

### Aesthetic Direction
延续项目现有的轻柔色彩和本地知识库工具感，以研究者工作台为核心而不是展示型 landing page。优先使用可读性高、信息组织清晰、便于长期停留的布局。避免过度科幻、炫技式 dark dashboard，也避免商务后台式僵硬表格感。

### Design Principles
1. 所有核心动作都应直接服务于“输入 -> 判断 -> 待办 -> 复盘”的研究闭环。
2. 页面优先承载真实工作内容，而不是解释功能本身。
3. 对量化和画像要给出可操作解释，避免只给分数不给依据。
4. 新区块要能与现有 profile / feed / timeline 视觉体系平滑衔接。
5. 对长期使用友好，减少一次性炫技和难以维护的复杂装饰。
